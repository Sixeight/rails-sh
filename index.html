                                                        rails-sh

A mini shell for Ruby on Rails 3 that provides a uniform and quick access to commands.

Install

$ gem install rails-sh

Usage

$ cd /path/to/rails3-app/
$ rails-sh

Examples

Print help: 

> help

Execute a rails command: 

> rails generate controller foo bar

Execute a rake task: 

> rake db:migrate

Execute a bundler command: 

> bundle install

Restart: 

> restart

Eval as ruby script: 

> eval 1 + 1

Run a shell command: 

> ! ls

Exit: 

> exit
